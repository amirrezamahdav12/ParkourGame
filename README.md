# ✨ Features

- 🏃 Fluid First Person Controller
- 🧗 Wall Running
- 🛝 Sliding
- 🚀 Double Jump
- 📦 Jump Pads
- 🏗️ Moving Platforms
- 🎮 Unity Input System
- 🌆 Cyberpunk Environment
- 💡 Dynamic Lighting
- 🔊 Environmental Audio

---

# 🏗 Architecture

```
Player
 ├── Input System
 ├── Movement Controller
 │    ├── Walking
 │    ├── Sprint
 │    ├── Jump
 │    ├── Double Jump
 │    ├── Slide
 │    └── Wall Run
 │
 ├── Camera Controller
 └── Interaction System
```

The movement system is built around modular components, making it easy to extend with mechanics like climbing, grappling, or vaulting.
